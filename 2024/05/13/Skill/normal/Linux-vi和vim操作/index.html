Posted on 2024-05-13 In Skill Word count in article: 531 Reading time ≈ 2 mins.

一般模式

移动光标

指令 描述
i 在当前光标所在字符的前面，转为编辑模式。
I 在当前光标所在行的行首转换为编辑模式。
a 在当前光标所在字符的后面，转为编辑模式。
A 在光标所在行的行尾，转换为编辑模式。
o 在当前光标所在行的下方，新建一行，并转为编辑模式。
O 在当前光标所在行的上方，新建一行，并转为编辑模式。
ngg 光标移至第 n 行的行首（ n 为数字,想要跳转的行）
nG 光标移至第 n 行的行尾（ n 为数字,想要跳转的行）
G 转至文件结尾
0 （数字零）光标移至当前行行首
$ 光标移至当前行行末
fx 搜索当前行中下一个出现字母 x 的地方
ctrl+f 屏幕向下翻一页，相当于下一页。
ctrl+b 屏幕向下翻一页，相当于下一页。

文件操作

指令 描述
cc 删除整行，并且修改整行内容。
dd 删除该行，不提供修改功能。
ndd 删除当前行向下 n 行。
x 删除光标所在的字符。
X 删除光标前面的一个字符。
yy 复制当前行
nyy 复制当前行及其下面 n 行。
p 粘贴最近复制的内容。
s 删除光标所在字符。
r 替换光标处字符。
u 撤销上一步操作

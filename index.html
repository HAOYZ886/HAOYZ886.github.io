Linux - BPF调试

Posted on 2026-01-16 In Skill Word count in article: 222 Reading time ≈ 1 mins.

BCC、bpftrace和ply

BCC 的全称是 BPF Compiler Collection ，它是 libbcc 和 libbpf 库的具体实现（又称为前端实现）。它提供了一个编写内核 BPF 程序的 C 语言环境（还有 C++ 、Lua 和 Python 等），还提供了 70 多个 BPF 工具，用于性能分析和排障工作。
bpftrace 也是 libbcc 和 libbpf 库的前端实现。它和 BCC 相比，它的优势是编写功能强大的单行程序和短小脚本。BCC 则是更适合开发复杂的脚本和后台程序，还能额外调用其它库做额外处理。
ply 可以理解为是轻量化的 bpftrace ，适合在嵌入式 Linux 环境下使用。

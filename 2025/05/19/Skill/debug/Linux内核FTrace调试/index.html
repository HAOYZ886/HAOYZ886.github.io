branch

用于追踪条件判断 if ，三者只能选其一：

  • CONFIG_BRANCH_PROFILE_NONE ：默认使能该配置，意为关闭追踪条件判断
  • CONFIG_PROFILE_ANNOTATED_BRANCHES ：追踪带有 likely/unlikely 的条件判断
  • CONFIG_PROFILE_ALL_BRANCHES ：追踪所有的条件判断，包括带有 likely/unlikely 的条件判断

filter
